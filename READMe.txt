                                     ---Rock Paper Scissors Game--

A fun and interactive Rock, Paper, Scissors game built with HTML, CSS, and JavaScript.

Features:
 -Play against the computer: Choose Rock, Paper, or Scissors and see who wins.
 -Score Tracking: Keeps track of wins, losses, and ties.
 -Custom Alerts: Stylish modal displays game results.
 -Responsive Design: Works well on all devices.
How to Play:
 -Open index.html in a web browser.
 -Click a button to choose Rock, Paper, or Scissors.
 -The computer will make a random choice.
 -See the result in the pop-up modal, including the updated score.
Customization:
 -Change Background: Update the background-image in the CSS.
 -Edit Styles: Modify button styles and modal appearance in the CSS.
